Installation

NPM

Utilisez NPM (Node Package Manager) pour installer les librairies JavaScript nécessaires.

Installer NodeJS + NPM

create-react-app

La commande create-react-app n'est plus recommandée pour créer un projet React.

Elle n'est plus documentée dans la doc officielle et il est maintenant recommandé d'utiliser React avec un framework tel que Next.js ou Remix.

Installation avec Vite

Vite est un outil de build ultra-rapide pour les applications web modernes. Il peut être utilisé pour créer des applications React.

Pour créer un projet avec Vite, il suffit de lancer la commande suivante :

npm create vite@latest

Il faut ensuite sélectionner l'option React puis JavaScript + SWC

L'utilisation de TypeScript serait également intéressante, mais cela nécessite l'apprentissage de ce langage.

Démarrer le serveur de développement

Une fois le projet créé, il suffit de se rendre dans le dossier du projet et de lancer la commande suivante pour installer les dépendances : 

npm install

Ensuite, pour démarrer le serveur de développement, il suffit de lancer la commande suivante :

npm run dev

Vite inclus un système HMR (Hot Module Replacement) qui permet de voir les changements en temps réel.

Attention au changement de contexte !

Dans l'exemple précédent, lorsque la fonction handleClick() sera appelée, un changement de contexte sera effectué.

Ainsi, il ne sera pas possible de modifier le state en appelant this.state() car la variable this aura pour valeur undefined.

Il faut alors effectuer un binding manuel directement lors de l'appel de la fonction :


 {/* Arrow function syntax */}

Ou bien effectuer un binding général dans le constructeur :

this.handleClick = this.handleClick.bind(this);

Mise en pratique

Nous allons maintenant ajouter dans notre application un nouveau composant ColorPicker qui nous permettra de changer la couleur de fond de l'application.

  1. Créer un nouveau composant permettant d'afficher un input de type color
  2. Identifier où sera situé le state
  3. Utiliser les props pour remonter l'événement de changement de couleur
Lorsque l'état d'un input est contrôlé par ReactJS, on parle de Controlled Component

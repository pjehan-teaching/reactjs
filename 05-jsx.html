JSX

JavaScript XML (JSX)

JSX est une extension du langage JavaScript.

Ce langage peut sembler similaire à un moteur de template (pug, mustache, EJS...) mais il permet de bénéficier de toute la puissance du langage JS.

Le code JSX permet de générer des éléments React.

Pourquoi JSX ?

ReactJS permet de découper son code en composants. Chaque composant peut contenir l'ensemble des informations nécessaires à son fonctionnement (UI, data, events...).

Le code source devient ainsi plus modulaire et réutilisable.

JSX vs Plain JavaScript

Lors de la compilation, Babel convertit le code JSX en code JavaScript.

Ainsi, les deux exemples ci-dessous produiront le même résultat :

return (<h1 className="hello">
    Hello world!
</h1>)
return React.createElement(
    'h1',
    {className: 'hello'},
    'Hello, world!'
)

A partir de ces éléments, React peut ensuite générer le DOM correspondant.

Les attributs sont écrit en camelCase !

Variables JavaScript

Du code JavaScript peut être inclu dans du code JSX en utilisant des accolades :

render(){
    const username = 'Pierre';
    return(
        <div className="App">
            <h1> Hello, {username}! </h1>
        </div>
    );
}

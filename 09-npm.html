NPM Link

La commande npm link permet d'effectuer un lien symbolique entre deux packages installés localement.

Cela peut permettre de tester un package avant de le publier sur NPM :

cd ./mon-package
npm link
cd ../mon-projet
npm link mon-package

Pour supprimer un lien symbolique :

npm unlink mon-package

Créer un widget

Vous allez maintenant créer un widget qui sera compatible avec le package react-realtime-dashboard

  1. Créez un projet test avec la commande create-react-app
  2. Créez un nouveau package avec la commande npm init (vous pouvez suivre ce tutoriel pour vous aider à configurer votre projet)
  3. Développez votre composant React

Publier un package du NPM

Une fois votre composant développé, vous pouvez maintenant le publier sur NPM :

  1. Créez un compte sur le internet de NPM ou utilisez la commande npm adduser
  2. Identifiez-vous en utilisant la commande npm login
  3. Créez un fichier .npmignore et définissez les dossiers/fichiers qui ne seront pas envoyés sur NPM (si il n'y a pas de .npmignore, le .gitignore est utilisé)
  4. Créez un fichier README.md
  5. Publiez votre package avec la commande npm publish

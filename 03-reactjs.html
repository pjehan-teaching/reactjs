ReactJS

Caractéristiques principales

  • Component based : Découpage du code source en composants
  • Virtual DOM : Utilisation d'un DOM virtuel ce qui permet d'optimiser les performances
  • JSX : Langage proche du HTML intégré directement dans les composants
  • Manual binding : Le binding avec les éléments du DOM doit se faire explicitement
  • Unidirection data-flow : Les données se propagent des composants parents vers les enfants
  • React Native : Permet de développer des applications mobiles
